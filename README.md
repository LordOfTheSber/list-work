# list-work

## Важные изменения
- Даты в формах путевых листов вводятся в формате **ДД.ММ.ГГГГ** (пример: `11.11.2000`).
- Можно ввести дату вручную или выбрать через кнопку `📅` рядом с полем.
- В таблице путевых листов добавлены горизонтальный и вертикальный скроллы.

## Сборка `.exe` (Windows)
1. Установите Python 3.10+.
2. Запустите `build_exe.bat` из корня проекта.
3. Готовый файл появится по пути `dist/fuel_logs_app.exe`.

Альтернатива: можно собирать через spec-файл `fuel_logs_app.spec`:
```bash
pyinstaller fuel_logs_app.spec
```
